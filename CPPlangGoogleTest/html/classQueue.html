Detailed Description

template<typename T>
class Queue< T >

Шаблонный класс Очереди (Queue).

Реализует структуру данных FIFO (First In, First Out) на основе односвязного списка. Элементы добавляются в "хвост" (back) и извлекаются из "головы" (front).
