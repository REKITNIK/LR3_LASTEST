Detailed Description

template<typename T>
class DoubleList< T >

Класс двусвязного списка.

Поддерживает вставку и удаление элементов с обоих концов за O(1), а также доступ и модификацию по индексу за O(N).

Template Parameters
TТип элементов списка. Должен быть копируемым и конструируемым по умолчанию.
Warning
Бинарная сериализация корректна только для тривиально копируемых типов (POD).
