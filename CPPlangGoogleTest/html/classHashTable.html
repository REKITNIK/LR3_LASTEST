Detailed Description

template<typename K, typename V>
class HashTable< K, V >

Шаблонный класс Хэш-таблицы (Hash Table).

Реализует словарь (ассоциативный массив) на основе массива списков (метод цепочек). При возникновении коллизий элементы добавляются в начало связного списка соответствующего бакета. Автоматически расширяется (rehashing) при превышении коэффициента заполнения.

Template Parameters
KТип ключа. Должен поддерживать сравнение (operator==) и хеширование (std::hash).
VТип значения.
