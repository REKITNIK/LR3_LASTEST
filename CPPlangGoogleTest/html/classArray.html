Detailed Description

template<typename T>
class Array< T >

Класс динамического массива с автоматическим изменением ёмкости.

Реализует последовательный контейнер с доступом по индексу. Управляет памятью вручную через new/delete.

Template Parameters
TТип элементов массива. Должен быть копируемым и конструируемым по умолчанию.
Warning
Бинарная сериализация корректна только для тривиально копируемых типов (POD).
