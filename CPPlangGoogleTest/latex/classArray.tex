\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class Array$<$ T $>$}
Класс динамического массива с автоматическим изменением ёмкости. 

Реализует последовательный контейнер с доступом по индексу. Управляет памятью вручную через new/delete.


\begin{DoxyTemplParams}{Template Parameters}
{\em T} & Тип элементов массива. Должен быть копируемым и конструируемым по умолчанию. \\
\hline
\end{DoxyTemplParams}
\begin{DoxyWarning}{Warning}
Бинарная сериализация корректна только для тривиально копируемых типов (POD). 
\end{DoxyWarning}
