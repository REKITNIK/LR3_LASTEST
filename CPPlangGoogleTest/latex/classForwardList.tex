\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class Forward\+List$<$ T $>$}
Класс односвязного списка. 

Реализует последовательный контейнер с эффективной вставкой/удалением в начале (O(1)). Доступ к произвольным элементам и вставка в конец выполняются за O(\+N).


\begin{DoxyTemplParams}{Template Parameters}
{\em T} & Тип элементов списка. Должен быть копируемым и конструируемым по умолчанию. \\
\hline
\end{DoxyTemplParams}
\begin{DoxyWarning}{Warning}
Бинарная сериализация корректна только для тривиально копируемых типов (POD). 
\end{DoxyWarning}
