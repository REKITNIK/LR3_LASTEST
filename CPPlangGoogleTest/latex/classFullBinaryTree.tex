\subsubsection*{template$<$typename T$>$\newline
class Full\+Binary\+Tree$<$ T $>$}
Шаблонный класс полного бинарного дерева (Full Binary Tree). 

Реализует структуру данных, где каждый узел имеет либо 0, либо 2 потомка. Вставка новых элементов происходит таким образом, чтобы дерево заполнялось равномерно, поддерживая свойство полноты.
