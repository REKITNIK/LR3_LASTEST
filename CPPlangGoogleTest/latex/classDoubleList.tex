\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class Double\+List$<$ T $>$}
Класс двусвязного списка. 

Поддерживает вставку и удаление элементов с обоих концов за O(1), а также доступ и модификацию по индексу за O(\+N).


\begin{DoxyTemplParams}{Template Parameters}
{\em T} & Тип элементов списка. Должен быть копируемым и конструируемым по умолчанию. \\
\hline
\end{DoxyTemplParams}
\begin{DoxyWarning}{Warning}
Бинарная сериализация корректна только для тривиально копируемых типов (POD). 
\end{DoxyWarning}
