\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class Stack$<$ T $>$}
Шаблонный класс Стека (\doxylink{classStack}{Stack}). 

Реализует структуру данных LIFO (Last In, First Out) на основе односвязного списка. Элементы добавляются на "{}вершину"{} и извлекаются с "{}вершины"{}.


\begin{DoxyTemplParams}{Template Parameters}
{\em T} & Тип хранимых данных. Должен иметь конструктор по умолчанию (для внутренних операций копирования). \\
\hline
\end{DoxyTemplParams}
